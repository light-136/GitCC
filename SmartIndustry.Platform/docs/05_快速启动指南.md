# SmartIndustry Platform — 快速启动指南

> 版本：1.0 | 日期：2026-05-26

## 1. 环境要求

| 组件 | 最低版本 | 推荐版本 |
|------|---------|---------|
| .NET SDK | 9.0 | 9.0.5+ |
| Visual Studio | 2022 17.8+ | 2022 17.11+ |
| 操作系统 | Windows 10 | Windows 11 |
| 内存 | 4 GB | 8 GB |
| 磁盘 | 500 MB | 1 GB |

## 2. 获取代码

```bash
# 克隆项目
git clone <repository-url>
cd SmartIndustry.Platform
```

## 3. 编译项目

```bash
# 还原 NuGet 包并编译
dotnet build

# 或使用 Visual Studio 打开解决方案
# 打开文件：SmartIndustry.Platform.sln
```

## 4. 运行测试

```bash
# 运行所有单元测试
dotnet test

# 预期输出：已通过! - 失败: 0，通过: 68
```

## 5. 启动应用

```bash
# 命令行启动
dotnet run --project src/SmartIndustry.UI/SmartIndustry.UI.csproj

# 或在 Visual Studio 中将 SmartIndustry.UI 设为启动项目，按 F5 运行
```

### 5.1 首次启动说明

1. 应用启动时会自动创建 SQLite 数据库文件 `smartindustry.db`
2. 默认以**仿真模式**运行，无需连接真实硬件
3. 主界面左侧为功能导航栏，右侧为功能页面内容区
4. 底部状态栏显示系统运行状态和版本信息

## 7. 开发指南

### 7.1 添加新功能模块

1. 在 `Domain/Interfaces/` 定义服务接口
2. 在 `Domain/Entities/` 添加实体（如需要）
3. 在 `Application/` 实现服务逻辑
4. 在 `Application/Services/ServiceCollectionExtensions.cs` 注册 DI
5. 在 `tests/` 编写对应的单元测试

### 7.2 添加新页面

1. 在 `UI/Views/` 创建页面子目录和 XAML 文件
2. 在 `UI/Services/PageFactory.cs` 添加页面映射
3. 在 `MainWindow.xaml` 添加导航按钮

### 7.3 连接真实硬件

替换 DI 注册中的仿真实现为真实驱动：

```csharp
// 在 AddHardware() 中：
```

## 8. 常见问题

**Q: 编译时出现 NU1603 警告怎么办？**
A: 这是 MQTTnet 版本差异警告，不影响功能，可安全忽略。

**Q: 启动后界面空白怎么办？**
A: 确认已安装 .NET 9.0 Desktop Runtime，可运行 `dotnet --info` 检查。

**Q: 如何查看运行日志？**
A: 日志文件自动保存在应用目录下的 `Logs/` 文件夹，按日期滚动。
