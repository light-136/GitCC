# SmartIndustry Platform — 项目规程报告

> 版本：1.0 | 日期：2026-05-26

## 1. 项目概述

**项目名称**：SmartIndustry Platform（智能工业平台）

**项目定位**：基于 C# / .NET 9.0 / WPF 的多模块工业智能控制平台，面向半导体、SMT、自动化组装等行业场景。

**开发周期**：2026年5月（Phase 1-4 已完成）

**技术栈**：
| 技术 | 版本 | 用途 |
|------|------|------|
| .NET | 9.0 | 运行时框架 |
| WPF | .NET 9 内置 | 桌面 UI |
| CommunityToolkit.Mvvm | 8.4.0 | MVVM 框架 |
| EF Core + SQLite | 9.0.5 | 数据持久化 |
| Serilog | 4.2.0 | 结构化日志 |
| MQTTnet | 4.3.7 | MQTT 通信 |
| xUnit + Moq | 2.9.3 / 4.20 | 单元测试 |
| LiveCharts2 | 2.0.0-rc3.3 | 数据可视化 |

## 2. 架构分层

```
┌──────────────────────────────────────────────────┐
│                SmartIndustry.UI                    │  表现层（WPF）
├──────────────────────────────────────────────────┤
│             SmartIndustry.Application              │  应用层（用例编排）
├──────────────────────────────────────────────────┤
│  SmartIndustry.Hardware  │ SmartIndustry.Infrastructure │  硬件抽象 / 基础设施
├──────────────────────────────────────────────────┤
│              SmartIndustry.Domain                   │  领域层（核心业务）
└──────────────────────────────────────────────────┘
```

**依赖规则**：上层依赖下层，Domain 层零外部依赖。

## 3. 模块完成情况

### 3.1 Domain 层 — ✅ 100% 完成
- 9 个实体（BaseEntity + 8 个业务实体）
- 15 个枚举（SystemEnums.cs）
- 18 个领域事件（DomainEvent.cs）
- 13 个服务接口 + 3 个仓储接口
- 3 个值对象（Position3D、MotionProfile、VisionRegion）
- 规约模式完整实现（Specification Pattern）

### 3.2 Infrastructure 层 — ✅ 100% 完成
- EF Core DbContext（软删除全局过滤、审计字段自动填充）
- 泛型仓储 + 2 个专用仓储（AlarmRepository、UserRepository）
- 进程内事件总线（弱引用、异常隔离、Lambda 订阅）
- Serilog 日志服务（双通道、环形缓冲区、实时订阅）
- 异步 TCP 客户端/服务端（粘包处理、自动重连、心跳）
- HTTP REST 客户端、MQTT 客户端
- 文件服务（安全根目录限制）、JSON 配置热重载

### 3.3 Hardware 层 — ✅ 100% 完成
- 30轴仿真运动控制卡（梯形速度曲线仿真）
- 梯形速度规划 + 7段 S曲线速度规划
- 直线/圆弧插补器
- 4优先级运动任务调度器
- 九点标定服务（仿射变换）
- 视觉管线（责任链模式）
- 纯数学图像处理（直方图、二值化、滤波、Sobel）
- 数字IO管理（边沿检测、消抖）
- 视觉-运动协同对位

### 3.4 Application 层 — ✅ 100% 完成（本次补全）
- 设备状态机（DeviceStateMachine）
- 报警服务（AlarmService）
- 配方管理服务（RecipeService）
- 用户认证与权限服务（UserService）
- 任务调度器（AppTaskScheduler）
- SECS/GEM 通信服务（SecsGemService）
- 自动化流程服务（AutomationService）
- DI 注册扩展方法

### 3.5 UI 层 — 🔶 60% 完成
- 主窗口框架 + 导航系统（✅ 完成）
- 9 个功能页面 XAML 骨架（✅ 完成）
- ViewModels / 数据绑定层（⏳ 待完善）

### 3.6 Tests 层 — ✅ 100% 完成（本次补全）

- 全部通过
