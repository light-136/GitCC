## 2. 模块依赖关系

```
    UI层
    │
    ├──▶ Application层
    │     ├──▶ Domain层 (模型、接口、枚举)
    │     └──▶ Infrastructure层 (事件总线、日志)
    │
    ├──▶ Hardware层
    │     ├──▶ Domain层
    │     └──▶ Infrastructure层 (事件总线)
    │
    └──▶ Infrastructure层
          └──▶ Domain层

    依赖规则：上层依赖下层，同层不互相依赖
    Domain层不依赖任何其他层
```

## 3. 核心模块交互

### 3.1 运动控制链路

```
MainViewModel
    │
    ▼
MotionViewModel ──▶ AxisManager ──▶ IMotionCard ──▶ IAxisController
    │                    │               │                │
    │              管理30轴          SimulationCard     模拟运动
    │              配置/分组          8轴/卡            梯形曲线
    │                    │
    │              MotionScheduler (生产者-消费者)
    │                    │
    └── UI状态刷新 ◀─── 轴状态更新 (200ms周期)
```

### 3.2 视觉系统链路

```
VisionViewModel ──▶ CameraManager ──▶ ICamera (SimulationCamera)
    │                                      │
    │                              打开/关闭/采集
    │                                      │
    └──▶ IVisionEngine (SimulationVisionEngine)
              │
              ├── TemplateMatchAsync()   模板匹配
              ├── BlobAnalysisAsync()    Blob分析
              ├── OcrRecognizeAsync()    OCR识别
              ├── FindMarkAsync()        Mark点定位
              ├── MeasureAsync()         尺寸测量
              └── DefectDetectAsync()    缺陷检测
```

### 3.3 视觉-运动协同链路

```
VisionMotionCoordinator
    │
    ├──▶ CameraManager.CaptureAsync()      采集图像
    │
    ├──▶ VisionEngine.FindMarkAsync()       检测Mark点
    │
    ├──▶ CoordinateTransformer              坐标转换
    │       │
    │       ├── PixelToWorld(px, py)         像素→机械
    │       │     wx = a*px + b*py + tx
    │       │     wy = c*px + d*py + ty
    │       │
    │       └── Calibrate9Point()           9点标定
    │             最小二乘法拟合仿射矩阵
    │
    └──▶ AxisManager.MoveRelativeAsync()    运动补偿
```

### 3.4 SECS/GEM通讯链路

```
SecsGemViewModel ──▶ ISecsGemService (SecsGemService)
    │                      │
    │               HsmsConnection (HSMS-SS传输层)
    │                      │
    │               ┌──────┴──────┐
    │               │ TCP Socket  │
    │               └─────────────┘
    │
    ├── S1F1/S1F2   在线确认
    ├── S1F13/S1F14  建立通讯
    ├── S2F41/S2F42  远程控制
    ├── S5F1/S5F2   报警上报
    └── S6F11/S6F12  事件报告

    GemEquipment (GEM合规层)
    ├── ControlStateMachine   (Online/Offline/Local/Remote)
    └── CommunicationStateMachine
```

### 3.5 报警系统链路

```
AlarmManager
    │
    ├── TriggerAlarm(code, level, message, source)
    │       │
    │       ├── 检查重复 (相同code不重复触发)
    │       ├── 添加到 ActiveAlarms
    │       ├── 添加到 AlarmHistory
    │       ├── 触发 AlarmTriggered 事件
    │       └── 通过 EventBus 发布 AlarmTriggeredEvent
    │
    ├── ClearAlarm(code, operatorName)
    │       │
    │       ├── 从 ActiveAlarms 移除
    │       ├── 更新历史记录 (ClearedAt, ClearedBy)
    │       └── 触发 AlarmCleared 事件
    │
    └── EventBus ──▶ MainViewModel (更新报警计数)
                 ──▶ SecsGemService (S5F1报警上报)
                 ──▶ StateMachine (状态转换到Alarm)
```

### 3.6 设备状态机

```
┌─────────┐  Initialize  ┌──────┐
│  Idle   │─────────────▶│ Init │
└────▲────┘              └──┬───┘
     │                 ┌────┴────┐
   Reset          StartAuto   Manual
     │            ┌───▼──┐  ┌──▼────┐
     │            │ Auto │  │Manual │
     │            └──┬───┘  └──┬────┘
     │          Alarm│      Alarm│
     │          ┌────▼─────────▼────┐
     └──────────│      Alarm       │
                └────────┬─────────┘
                  EmergencyStop
                ┌────────▼─────────┐
                │  EmergencyStop   │
                └──────────────────┘

Guard条件: 报警状态下不允许初始化
转换动作: 状态变更通过EventBus通知全系统
```

## 4. 事件总线通讯图

```
EventBus (发布/订阅中心)
    │
    ├── AlarmTriggeredEvent
    │     发布者: AlarmManager
    │     订阅者: MainViewModel, SecsGemService, StateMachine
    │
    ├── StateChangedEvent
    │     发布者: DeviceStateMachine
    │     订阅者: MainViewModel, SecsGemService
    │
    ├── LogEvent
    │     发布者: LogService
    │     订阅者: MainViewModel (日志面板)
    │
    ├── UserChangedEvent
    │     发布者: UserManager
    │     订阅者: MainViewModel (用户显示)
    │
    └── RecipeChangedEvent
          发布者: RecipeManager
          订阅者: 生产流程模块
```

## 6. 线程模型

```
IndustrialTaskScheduler
│
├── RealTime 优先级 (独立线程, LongRunning)
│   └── 运动控制更新 (10ms周期)
│
├── High 优先级 (独立线程, LongRunning)
│   └── IO信号处理 (10-50ms周期)
│
├── Normal 优先级 (ThreadPool)
│   ├── 轴状态刷新 (200ms周期)
│   └── 数据采集 (100-500ms周期)
│
└── Background 优先级 (ThreadPool)
    ├── UI时钟刷新 (1000ms周期)
    └── 日志写入 (500-5000ms周期)

UI线程: Dispatcher.Invoke() 确保线程安全
```

## 7. 数据流向

```
传感器/相机 ──▶ 硬件驱动 ──▶ 事件总线 ──▶ ViewModel ──▶ UI绑定
                                │
                                ├──▶ 报警系统 ──▶ SECS/GEM上报
                                ├──▶ 日志系统 ──▶ Serilog文件
                                └──▶ 状态机 ──▶ 状态转换

用户操作 ──▶ Command ──▶ ViewModel ──▶ Service ──▶ 硬件驱动
                                         │
                                         └──▶ 事件总线 ──▶ 其他模块
```
