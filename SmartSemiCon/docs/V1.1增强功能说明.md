# SmartSemiCon V1.1 增强功能说明

## 版本变更概述

V1.1 对视觉系统和运动控制模块进行了增强，新增可视化模拟演示功能，使用户可以直观观察完整的半导体设备工艺流程。

---

## 一、运动控制增强 (MotionViewModel)

### 新增功能

| 功能 | 命令 | 说明 |
|------|------|------|
| 直线插补 | `LinearInterpolationCommand` | 选择2轴进行直线插补运动演示 |
| 圆弧插补 | `ArcInterpolationCommand` | 选择2轴画1/4圆弧（18段逼近） |
| 循环运动 | `StartCycleCommand` / `StopCycleCommand` | 选中轴在起点-终点间自动往返 |
| 点位示教 | `TeachCurrentPositionCommand` | 保存当前轴位置到示教列表 |
| 执行示教序列 | `RunTeachSequenceCommand` | 按顺序移动到所有示教位置 |
| 轨迹记录 | 自动 | 每次运动完成记录(时间,轴ID,位置) |
| 全部急停 | `EmergencyStopAllCommand` | 新增到操作面板 |

### 新增参数

| 参数 | 默认值 | 说明 |
|------|--------|------|
| InterpAxisX | 0 | 插补X轴编号 |
| InterpAxisY | 1 | 插补Y轴编号 |
| InterpDistance | 50mm | 插补距离 |
| CycleStart | 0mm | 循环起始位置 |
| CycleEnd | 100mm | 循环终止位置 |
| CycleCount | 3 | 循环次数 |

## 二、视觉系统增强 (VisionViewModel)

### 新增功能

| 功能 | 命令 | 说明 |
|------|------|------|
| 连续检测 | `StartContinuousCommand` / `StopContinuousCommand` | 自动循环采集+检测，实时更新统计 |
| 多相机同步 | `CaptureAllCamerasCommand` | 4台相机同时采集，并行检测 |
| 相机参数 | `ApplyCameraParamsCommand` | 调节曝光时间和增益 |
| 结果详情 | 自动 | 检测结果显示ExtraData详细信息 |

### 新增参数

| 参数 | 默认值 | 说明 |
|------|--------|------|
| ContinuousInterval | 500ms | 连续检测间隔 |
| ExposureTime | 3000us | 相机曝光时间 |
| Gain | 1.0 | 相机增益 |

---

## 三、视觉-运动协同演示 (MainViewModel)

### 新增功能

| 功能 | 命令 | 说明 |
|------|------|------|
| 9点标定演示 | `RunCalibrationDemoCommand` | 自动执行标定流程，显示RMS误差和标定点数 |
| 视觉对位演示 | `RunAlignDemoCommand` | 拍照→检测→坐标转换→运动补偿 全流程 |
| 生产流程模拟 | `RunProductionDemoCommand` | 7步完整工艺：上料→定位→检测→搬运→下料→复位 |

### 生产流程模拟详细步骤

```
步骤1/7: 上料位移动
  └─ 搬运机器人移动到上料位 (X=100, Y=50)

步骤2/7: 搬运到检测位
  └─ Z轴下降取料→上升→X轴搬运到检测位 (X=200)

步骤3/7: 视觉定位
  └─ 相机采集→FindMark算法检测Mark点坐标

步骤4/7: 对位补偿
  └─ 根据视觉结果进行微调运动

步骤5/7: 外观检测
  └─ 顶部相机采集→DefectDetect缺陷检测

步骤6/7: 搬运到下料位
  └─ 搬运到下料位 (X=300)→Z轴下降放料

步骤7/7: 复位
  └─ 所有轴回到待机位 (0, 0)
```

## 四、UI 变更

### Dashboard 页面
- 新增"视觉-运动协同演示"区域
- 包含：9点标定演示、视觉对位演示、生产流程模拟 三个按钮
- 实时显示：演示状态（DemoStatus）和标定信息（CalibrationInfo）

### Motion 页面
- 新增"增强功能面板"，分三栏：
  - 插补运动：X轴/Y轴/距离选择，直线/圆弧插补按钮
  - 循环运动：起点/终点/次数设置，开始/停止按钮，状态显示
  - 点位示教：示教/执行/清空按钮，示教点计数
- 轴状态表格与轨迹记录双栏显示

### Vision 页面
- 新增"增强功能面板"，分两栏：
  - 连续检测：间隔设置，启动/停止按钮，状态显示
  - 相机参数：曝光时间/增益设置，应用参数按钮
- 新增"全部相机采集"按钮
