- **.NET 桌面开发** — WPF 开发所需

## 4. 运行项目

### 4.1 通过命令行运行

```bash
dotnet run --project src/SmartSemiCon.UI/SmartSemiCon.UI.csproj
```

### 4.2 通过 Visual Studio 运行

1. 双击 `SmartSemiCon.sln` 打开解决方案
2. 在解决方案资源管理器中，右键 `SmartSemiCon.UI` → **设为启动项目**
3. 按 `F5` 或点击 **启动调试**

### 4.3 启动过程

应用启动时自动执行以下初始化：

```
1. 配置 Serilog 日志 → Logs/SmartSemiCon-{date}.log
2. 注册 DI 容器服务（事件总线、日志、报警、用户、配方等）
3. 初始化运动控制 → 配置 30 个模拟轴（4 张模拟控制卡）
4. 初始化视觉系统 → 配置 4 台模拟相机
5. 配置视觉-运动协同标定参数（默认 1像素 = 0.01mm）
6. 显示主窗口
```

## 7. 基本操作流程

### 7.1 设备初始化流程

```
步骤1: 登录系统
  └→ 点击 User 页面 → 输入 admin/admin123 → 登录

步骤2: 初始化设备
  └→ 点击 "初始化设备" 按钮
  └→ 系统自动：初始化控制卡 → 使能所有轴
  └→ 设备状态：Idle → Init

步骤3: 进入运行模式
  └→ 点击 "自动运行" → 设备状态变为 Auto
  └→ 或点击 "手动模式" → 设备状态变为 Manual
```

### 7.2 运动控制操作

```
步骤1: 进入 Motion 页面
步骤2: 选择一个轴（如 "搬运-X轴"）
步骤3: 使能轴（Servo On）
步骤4: 执行运动:
  - JOG:  按住 "正方向/负方向" 按钮，松开停止
  - 绝对移动: 输入目标位置 → 点击 "移动"
  - 回原点: 点击 "Home"
步骤5: 急停: 点击 "急停" 按钮立即停止所有轴
```

### 7.3 视觉系统操作

```
步骤1: 进入 Vision 页面
步骤2: 选择相机（如 "顶部相机"）
步骤3: 打开相机（Open）
步骤4: 采集图像（Capture）
步骤5: 选择视觉算法（模板匹配/Blob/OCR/...）
步骤6: 执行检测 → 查看结果（坐标、分数、耗时）
```

### 7.4 紧急操作

| 操作 | 触发方式 | 系统响应 |
|------|---------|---------|
| **急停** | 主界面急停按钮 | 所有轴立即停止运动，状态 → EmergencyStop |
| **报警复位** | 主界面复位按钮 | 清除所有活跃报警，状态 → Idle |
| **停止** | 主界面停止按钮 | 设备安全停止，状态 → Idle |

## 9. 常见问题

### 9.1 编译错误："找不到 .NET 9.0 SDK"

**原因**：未安装 .NET 9.0 SDK  
**解决**：从 https://dotnet.microsoft.com/download/dotnet/9.0 下载安装

### 9.2 运行时错误："未能加载 WPF 组件"

**原因**：项目目标框架为 `net9.0-windows`，需要 Windows 平台  
**解决**：确认在 Windows 10/11 上运行

### 9.3 NuGet 还原失败

**解决**：
```bash
# 清理缓存后重新还原
dotnet nuget locals all --clear
dotnet restore SmartSemiCon.sln
```

### 9.4 UI 界面无响应

**原因**：长时间耗时操作在 UI 线程执行  
**解决**：所有耗时操作已设计为异步（async/await），如遇到此问题请提交 issue
