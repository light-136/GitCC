## 3. 设备总览仪表盘 (Dashboard)

Dashboard 页面提供设备运行的全局视图：

### 3.1 显示内容

| 区域 | 显示信息 |
|------|---------|
| 设备状态 | 当前设备运行状态（带颜色指示） |
| 轴概况 | 已配置轴数 (30)，各轴运行状态汇总 |
| 相机概况 | 已配置相机数 (4)，相机连接状态 |

| 按钮 | 功能 | 前置条件 |
|------|------|---------|
| 初始化设备 | 初始化控制卡 + 使能所有轴 | 设备处于 Idle 且无活跃报警 |
| 自动运行 | 进入自动运行模式 | 设备已初始化 |
| 停止 | 停止运行回到 Idle | 设备正在运行 |

### 4.1 界面布局

└──────────────┴────────────────────────────────────────┘
```

### 4.2 操作步骤

#### 4.2.1 使能轴 (Servo On)

1. 在轴列表中选择目标轴
2. 点击 **"使能"** 按钮
3. 等待使能完成（约 50ms）
4. 状态显示从 `NotReady` 变为 `Ready`

#### 4.2.2 JOG 点动操作

| 操作 | 说明 |
|------|------|
| **JOG+** | 按住按钮，轴向正方向持续运动；松开停止 |
| **JOG-** | 按住按钮，轴向负方向持续运动；松开停止 |

注意事项：
- 轴必须已使能 (Servo On)
- 触碰软限位时自动停止
- JOG 速度可在设置中调整

#### 4.2.3 绝对定位运动

1. 在 **"目标位置"** 输入框中输入目标坐标（单位 mm）
2. 在 **"速度"** 输入框中输入运动速度（单位 mm/s）
3. 点击 **"绝对移动"** 按钮
4. 观察位置实时更新，直到到达目标位置
5. 状态从 `Moving` 变为 `Done`

#### 4.2.4 相对定位运动

1. 输入相对移动距离（正值向正方向，负值向负方向）
2. 输入运动速度
3. 点击 **"相对移动"** 按钮

#### 4.2.5 回原点 (Home)

1. 选择目标轴
2. 点击 **"回原"** 按钮
3. 轴自动以 HomeVelocity (默认 10mm/s) 向原点方向运动
4. 到达原点后，状态显示 `IsHomed: YES`

#### 4.2.6 急停操作

| 按钮 | 范围 | 效果 |
|------|------|------|
| 单轴停止 | 当前选中轴 | 立即停止当前轴运动 |
| 全局急停 | 所有 30 轴 | 立即停止所有轴运动 + 设备进入急停状态 |

### 4.3 运动参数说明

| 参数 | 默认值 | 单位 | 说明 |
|------|-------|------|------|
| 最大速度 | 200 | mm/s | 轴允许的最大运行速度 |
| 最大加速度 | 1000 | mm/s^2 | 最大加速度 |
| 最大减速度 | 1000 | mm/s^2 | 最大减速度 |
| 正向软限位 | +500 | mm | 正方向最大位置 |
| 负向软限位 | -500 | mm | 负方向最大位置 |
| 回原速度 | 10 | mm/s | 回原点运动速度 |

```

### 5.2 操作步骤

#### 5.2.1 打开相机

1. 在相机列表中选择目标相机
2. 点击 **"打开"** 按钮
3. 相机状态变为 "已打开"

#### 5.2.2 图像采集

| 操作 | 说明 |
|------|------|
| **采集** | 触发一次软件采集，获取一帧图像 |
| **连续采集** | 持续采集图像流，实时预览 |

#### 5.2.3 调整相机参数

| 参数 | 范围 | 说明 |
|------|------|------|
| 曝光时间 | 100~100000 us | 控制图像亮度 |
| 增益 | 0~20 | 信号放大倍数 |
| 触发模式 | 软触发/硬触发/连续/编码器 | 采集触发方式 |

#### 5.2.4 执行视觉检测

1. 确保已采集图像
2. 选择视觉算法：
   - **模板匹配** — 在图像中查找模板位置
   - **Blob 分析** — 提取连通区域特征
   - **OCR 识别** — 识别图像中的文字
   - **Mark 定位** — 查找标记点位置
   - **尺寸测量** — 测量距离/角度
   - **缺陷检测** — 检测产品表面缺陷
3. 点击 **"执行检测"**
4. 查看检测结果面板

### 5.3 检测结果说明

| 字段 | 说明 |
|------|------|
| IsSuccess | 检测是否成功 |
| PixelX/Y | 检测目标的像素坐标 |
| Angle | 检测目标的角度（度） |
| Score | 匹配分数（0~1，越高越好） |
| WorldX/Y | 经标定转换后的机械坐标（mm） |
| ProcessTimeMs | 算法处理耗时（ms） |
| ExtraData | 算法特定的扩展数据 |

## 6. SECS/GEM 通讯使用 (SecsGem)

### 6.1 界面布局

```
┌───────────────────────────────────────────────────────┐
│                   SECS/GEM 通讯面板                    │
├──────────────────┬────────────────────────────────────┤
│  连接状态         │  消息收发                            │
│                  │                                    │
│  HSMS: Selected  │  发送: S[  ]F[  ] Body:[        ] │
│  GEM:  OnlineRemote│ [发送] [发送并等待回复]            │
│  Comm: Communicating│                                 │
│                  │  接收消息列表:                       │
│  Host: 127.0.0.1 │  ┌────────────────────────────────┐│
│  Port: 5000      │  │ 12:00:01 S1F1 Are You There   ││
│                  │  │ 12:00:02 S1F2 Reply            ││
│  [连接] [断开]    │  │ 12:00:05 S6F11 Event Report   ││
│  [上线] [离线]    │  │ ...                            ││
│                  │  └────────────────────────────────┘│
│  设备ID: [1]     │                                    │
│  T3超时: [45]s   │  事件/报警:                         │
│                  │  [触发事件] EventID:[  ]            │
│                  │  [设置报警] AlarmID:[  ] Text:[  ]  │
│                  │  [清除报警] AlarmID:[  ]            │
└──────────────────┴────────────────────────────────────┘
```

### 6.2 连接操作

1. 输入 Host IP 和 Port
2. 点击 **"连接"** → HSMS TCP 连接
3. 自动发送 Select.req → 建立 HSMS 会话
4. 连接成功后 HSMS 状态变为 `Selected`
5. 点击 **"上线"** → 发送 S1F13 → GEM 状态变为 `OnlineRemote`

### 6.3 消息收发

| 操作 | 说明 |
|------|------|
| 发送 | 发送指定 SxFy 消息，不等待回复 |
| 发送并等待回复 | 发送消息并等待回复（T3 超时） |
| 触发事件 | 发送 S6F11（Event Report Send） |
| 设置报警 | 发送 S5F1（Alarm Report Send） |
| 清除报警 | 发送 S5F1（报警清除） |

### 6.4 常用 SECS 消息

| Stream/Function | 名称 | 方向 | 用途 |
|----------------|------|------|------|
| S1F1/S1F2 | Are You There | 双向 | 在线检测 |
| S1F13/S1F14 | Establish Comm | E→H | 建立通讯 |
| S5F1/S5F2 | Alarm Report | E→H | 报警上报 |
| S6F11/S6F12 | Event Report | E→H | 事件上报 |

## 9. 系统设置 (Settings)

### 9.1 可配置参数

| 分类 | 参数 | 默认值 | 说明 |
|------|------|-------|------|
| 通讯 | TCP 重连间隔 | 3000ms | 断线自动重连的间隔 |
| 通讯 | 心跳间隔 | 5000ms | 心跳包发送间隔 |
| 通讯 | 接收缓冲区 | 8192B | TCP 接收缓冲区大小 |
| SECS | T3 超时 | 45s | 等待回复超时 |
| SECS | T5 超时 | 10s | 连接分离超时 |
| SECS | T6 超时 | 5s | 控制事务超时 |
| SECS | 设备 ID | 1 | HSMS Session ID |
| 日志 | 日志级别 | Debug | 最低日志输出级别 |

### 10.3 登录操作

1. 在 **用户名** 输入框输入账号
2. 在 **密码** 输入框输入密码
3. 点击 **"登录"** 按钮
4. 登录成功后，状态栏显示当前用户名和角色
5. 操作界面根据权限级别显示/隐藏功能按钮

### 10.4 密码安全

- 密码使用 SHA256 算法哈希后存储
- 系统不保存明文密码
- 登录验证对比密码哈希值
