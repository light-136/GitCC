# CppMasterClass 模块划分说明

## 模块总览

项目共 11 个源文件，1 个入口 + 10 个功能模块：

```
src/
├── main.cpp                    入口：菜单系统
├── 01_basics.cpp              层级1：语言基础
├── 02_pointers.cpp            层级1：内存基础
├── 03_oop.cpp                 层级2：面向对象
├── 04_templates.cpp           层级2：泛型编程
├── 05_stl.cpp                 层级3：标准库
├── 06_memory.cpp              层级3：资源管理
├── 07_concurrency.cpp         层级4：并发编程
├── 08_modern_cpp.cpp          层级4：现代特性
├── 09_design_patterns.cpp     层级5：工程实践
└── 10_real_world.cpp          层级5：实战应用
```

## 各模块详细说明

### main.cpp — 菜单入口
- **职责**: 展示菜单、接收用户输入、分发到各模块
- **数据结构**: `MenuItem { id, name, function<void()> }`
- **依赖**: 所有模块的 `run_xxx()` 函数声明

### 01_basics.cpp — 基础语法
- **职责**: 展示 C++ 类型系统、控制流、字符串操作
- **关键知识点**:
  - `sizeof` 各类型大小
  - `auto` 类型推导
  - `constexpr` 编译期计算（阶乘）
  - `enum class` 强类型枚举
  - C++17 `if` 初始化语句
  - 字符串拼接/查找/截取
  - `static_cast` 类型转换
- **导出函数**: `void run_basics()`

### 02_pointers.cpp — 指针与引用
- **职责**: 展示 C++ 内存模型的核心概念
- **关键知识点**:
  - 原始指针、取地址、解引用
  - 指针算术（数组遍历）
  - `new/delete` 堆内存管理
  - `void*` 通用指针
  - 引用 vs 指针
  - `nullptr`
  - `unique_ptr/shared_ptr/weak_ptr` 概述
- **导出函数**: `void run_pointers()`

### 03_oop.cpp — 面向对象
- **职责**: 展示 C++ 的 OOP 体系
- **关键类型**:
  - `Vector2D` — 演示构造/析构/拷贝/移动/运算符重载/友元
  - `Shape`(抽象) → `Circle/Rectangle/Triangle` — 虚函数多态
  - `DataPoint` — 多重继承（`Printable` + `Serializable`）
- **导出函数**: `void run_oop()`

### 04_templates.cpp — 模板编程
- **职责**: 展示 C++ 泛型编程和元编程
- **关键知识点**:
  - 函数模板 `maximum<T>`
  - 类模板 `SimpleStack<T, MaxSize>`
  - 完全特化 `type_name<int>`
  - 可变参数模板 `print_all(args...)`
  - 折叠表达式 `sum_all(args...)`
  - SFINAE `enable_if`
  - C++20 `concepts`（条件编译）
  - 编译期斐波那契 `Fibonacci<N>`
- **导出函数**: `void run_templates()`

### 05_stl.cpp — STL 容器与算法
- **职责**: 展示标准库容器和算法的使用与性能
- **覆盖容器**: vector, list, map, set, unordered_map
- **覆盖算法**: sort, binary_search, minmax_element, accumulate, transform, count_if, remove_if, for_each
- **性能对比**: 10万元素插入/排序/查找的微秒级计时
- **导出函数**: `void run_stl()`

### 06_memory.cpp — 内存管理
- **职责**: 展示 C++ 资源管理的最佳实践
- **关键类型**:
  - `FileHandle` — RAII 模式演示（自动关闭文件）
  - `TrackedObject` — 带内存追踪的资源类
  - `Node` — 循环引用演示
- **关键知识点**: RAII、unique_ptr 独占、shared_ptr 共享、weak_ptr 打破循环、移动语义性能
- **导出函数**: `void run_memory()`

### 07_concurrency.cpp — 并发编程
- **职责**: 展示 C++ 多线程编程
- **关键类型**:
  - `ThreadSafeCounter` — mutex 保护的计数器
  - `ProducerConsumer` — condition_variable 的生产者-消费者
  - `ThreadPool` — 4 线程的简易线程池
- **关键知识点**: thread、mutex/lock_guard、atomic、condition_variable、future/async、线程池
- **导出函数**: `void run_concurrency()`

### 08_modern_cpp.cpp — 现代 C++ 特性
- **职责**: 展示 C++11/14/17/20 的现代语法
- **关键知识点**:
  - Lambda（基本/捕获/泛型）
  - auto/decltype
  - 结构化绑定 `auto [a, b] = pair`
  - `if constexpr`（编译期分支）
  - `std::optional/variant/any`
  - `std::string_view`（零拷贝）
  - `std::initializer_list`
- **导出函数**: `void run_modern_cpp()`

### 09_design_patterns.cpp — 设计模式
- **职责**: 用 C++ 实现常见设计模式
- **模式列表**:
  - 工厂方法 — `SensorFactory::create(type)`
  - 单例 — `SystemConfig::instance()`（C++11 线程安全）
  - 观察者 — `EventBus` 发布-订阅
  - 策略 — `MotionStrategy` 接口 + 3 种运动策略
  - CRTP — `Loggable<Derived>` 静态多态
- **导出函数**: `void run_design_patterns()`

### 10_real_world.cpp — 实战：运动控制器
- **职责**: 用纯 C++ 模拟半导体设备搬运流程
- **关键类型**:
  - `SimAxis` — 模拟轴（梯形速度规划、多线程安全）
  - `DeviceFSM` — 设备状态机（Idle→Init→Running）
- **模拟流程**: 初始化→使能→回原→搬运（取料→运输→放料）→复位
- **与 SmartSemiCon 的对应关系**: 概念和架构与 SmartSemiCon 的 SimulationAxisController 一致
- **导出函数**: `void run_real_world()`

## 模块依赖关系

```
main.cpp ──调用──▶ 所有 run_xxx() 函数
各模块之间无依赖关系（完全独立）
所有模块仅依赖 C++ 标准库
```
