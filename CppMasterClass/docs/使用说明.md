# CppMasterClass 使用说明

## 环境准备

### Windows
1. 安装 MinGW-w64 (推荐通过 MSYS2)
   ```
   pacman -S mingw-w64-x86_64-gcc
   ```
2. 或安装 Visual Studio 2019+ (自带 MSVC 编译器)

### Linux
```bash
sudo apt install g++   # Ubuntu/Debian
sudo yum install gcc-c++  # CentOS/RHEL
```

### macOS
```bash
xcode-select --install  # 安装 Clang
```

## 编译

### 方式一：直接编译（最简单）
```bash
# 进入项目目录
cd CppMasterClass

# g++ 编译
g++ -std=c++20 -O2 src/*.cpp -o CppMasterClass

# MSVC 编译
cl /std:c++20 /EHsc /O2 src\*.cpp /Fe:CppMasterClass.exe
```

### 方式二：CMake 构建
```bash
mkdir build && cd build
cmake ..
cmake --build .
```

## 运行

运行程序后进入交互式菜单：

```
╔══════════════════════════════════════════════════════╗
║        C++ 完整语言学习项目  (CppMasterClass)       ║
╠══════════════════════════════════════════════════════╣
║  1. 基础语法                                        ║
║  2. 指针与引用                                      ║
║  3. 面向对象                                        ║
║  ...                                                ║
║  10. 实战                                           ║
║  0. 运行全部模块                                    ║
║  q. 退出                                            ║
╚══════════════════════════════════════════════════════╝
```

- 输入 `1`~`10` 运行指定模块
- 输入 `0` 运行全部 10 个模块
- 输入 `q` 退出程序

## 关键知识点速查

### C++ 内存模型
```
栈(Stack): 局部变量，自动管理，LIFO
堆(Heap):  new/delete，手动管理 → 建议用智能指针
静态区:    全局变量、static变量
代码区:    程序指令
```

### 智能指针选择
```
unique_ptr: 独占所有权，不可拷贝，最轻量
shared_ptr: 共享所有权，引用计数，有开销
weak_ptr:   弱引用，不增加引用计数，打破循环
```

### 多线程同步
```
mutex + lock_guard: 最基本的互斥锁
atomic:             无锁原子操作，适合简单计数
condition_variable: 等待-通知机制
future/async:       异步任务返回值
```
