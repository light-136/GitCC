# CppMasterClass 项目概要说明

## 项目名称
C++ 完整语言学习项目 (CppMasterClass)

## 项目目的
系统性展示 C++ 语言的所有核心特性，通过 10 个可独立运行的模块，从基础语法到实战应用，每个模块都有实际的运行输出结果。

## 技术栈
- **语言**: C++20 (向下兼容 C++17)
- **编译器**: GCC 10+ / Clang 12+ / MSVC 2019+
- **构建系统**: CMake 3.20+ (可选，也支持直接 g++ 编译)
- **依赖**: 仅使用 C++ 标准库，无外部依赖

## 模块概述

| 编号 | 模块 | 源文件 | 核心知识点 |
|------|------|--------|-----------|
| 01 | 基础语法 | `01_basics.cpp` | sizeof、auto、const/constexpr、强枚举、C++17 if初始化 |
| 02 | 指针与引用 | `02_pointers.cpp` | 原始指针、指针算术、new/delete、引用、nullptr、智能指针概述 |
| 03 | 面向对象 | `03_oop.cpp` | 类生命周期、拷贝/移动语义、多态、抽象类、运算符重载、友元、多重继承 |
| 04 | 模板编程 | `04_templates.cpp` | 函数/类模板、特化、可变参数模板、折叠表达式、SFINAE、C++20 concepts、编译期斐波那契 |
| 05 | STL | `05_stl.cpp` | vector/list/map/set/unordered_map、算法(sort/find/transform)、性能对比 |
| 06 | 内存管理 | `06_memory.cpp` | RAII模式、unique_ptr/shared_ptr/weak_ptr、移动语义、完美转发、内存追踪 |
| 07 | 并发编程 | `07_concurrency.cpp` | thread、mutex、atomic、condition_variable、future/async、线程池 |
| 08 | 现代C++ | `08_modern_cpp.cpp` | lambda、auto/decltype、结构化绑定、if constexpr、optional/variant/any、string_view |
| 09 | 设计模式 | `09_design_patterns.cpp` | 工厂方法、单例(线程安全)、观察者、策略、CRTP(静态多态) |
| 10 | 实战 | `10_real_world.cpp` | 模拟运动控制器：多轴并行、梯形速度规划、状态机、搬运流程 |

## 运行方式
```bash
# 编译
g++ -std=c++20 -O2 src/*.cpp -o CppMasterClass

# 运行(交互式菜单)
./CppMasterClass

# 选择模块编号运行，或输入 0 运行全部
```

## 与 SmartSemiCon 的关系
模块 10 (实战) 使用纯 C++ 实现了与 SmartSemiCon 概念一致的运动控制器模拟，包括：
- 多轴并行运动（std::thread）
- 梯形速度规划（加速-匀速-减速）
- 设备状态机（Idle → Init → Running）
- 搬运流程模拟（取料-搬运-放料）

展示了 C++ 在半导体设备控制领域的应用能力。
