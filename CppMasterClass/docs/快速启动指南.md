# CppMasterClass 快速启动指南

## 1. 编译（三选一）

### g++ 直接编译（推荐）
```bash
cd CppMasterClass
g++ -std=c++20 -O2 src/*.cpp -o CppMasterClass
```

### MSVC 编译
```cmd
cd CppMasterClass
cl /std:c++20 /EHsc /O2 src\*.cpp /Fe:CppMasterClass.exe
```

### CMake 构建
```bash
mkdir build && cd build
cmake ..
cmake --build .
```

## 2. 运行

```bash
./CppMasterClass       # Linux/macOS
.\CppMasterClass.exe   # Windows
```

## 3. 操作

程序启动后显示菜单：
- 输入 `1`~`10` → 运行对应模块
- 输入 `0` → 运行全部 10 个模块
- 输入 `q` → 退出

## 4. 推荐学习路径

```
入门 → 01 基础 → 02 指针
进阶 → 03 OOP → 04 模板
高级 → 05 STL → 06 内存 → 07 并发
精通 → 08 现代C++ → 09 设计模式 → 10 实战
```

## 5. 常见问题

| 问题 | 解决方案 |
|------|---------|
| `g++: command not found` | 安装 MinGW-w64 或 GCC |
| `concepts` 相关报错 | 换用 `-std=c++17` 编译（concepts 自动跳过） |
| 中文乱码 | Windows 终端执行 `chcp 65001`（切换 UTF-8） |
