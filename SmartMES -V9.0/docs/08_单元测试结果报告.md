## 3. 过程问题与修复

### 问题

在首次执行中，`GCodeCommandTests.Parse_BlankLine_ShouldKeepUnknownAndNoAxis` 失败。

### 根因

`GCodeCommand.Type` 默认值未显式设为 `Unknown`，空行解析时保留了枚举默认值 `G0`。

### 修复

已在 `TenAxisController.cs` 中将 `GCodeCommand.Type` 默认值调整为：

```csharp
public GCodeType Type { get; set; } = GCodeType.Unknown;
```

修复后重新执行测试，全部通过。
