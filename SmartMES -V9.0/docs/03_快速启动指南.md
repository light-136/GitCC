## 快速启动

```powershell
# 1. 进入项目目录
cd 'D:\软件\Cursor\文档存放\代码区\SmartMES'

# 2. 构建
dotnet build SmartMES.sln

# 3. 运行
dotnet run --project SmartMES.UI
# 或直接运行编译产物
.\SmartMES.UI\bin\Debug\net9.0-windows\SmartMES.UI.exe
```

## 使用示例

### 数据库功能
1. 左栏点击「🗄 数据库」
2. 选择数据库类型（默认SQLite）
3. 点击「🔌 连接」→ 自动建表+生成60条仿真数据
4. 点击「📊 统计」→ 查看工单良品率统计
5. 点击「CSV/JSON/Excel」→ 选择保存路径导出

### 运动控制中心
1. 左栏点击「🎯 运动控制中心」
2. 点击「▶ 随机移动」→ 3轴同时运动
3. 点击「⏸ 全部暂停」→ 暂停所有轴
4. 点击「🛑 急停」→ 立即停止所有运动
5. 单轴「移至」输入框输入目标位置，点击「Go」

### 10轴版面切换
1. 左栏点击「🔢 10轴版面切换」
2. 点击「📄 载入示例」→ 加载内置五轴铣削程序
3. 点击「▶ 执行程序」→ 实时观察10轴运动
4. 点击「5×2布局 / 2×5布局」→ 按屏幕或工位习惯切换版面
5. 可直接编辑G代码文本框，自定义程序

### 视觉检测
1. 左栏点击「👁 视觉检测样例」
2. 点击「📷 启动摄像头」→ 8fps模拟帧刷新
3. 点击「⚠ 生成缺陷图」→ 强制产生含缺陷图像
4. 点击「🔍 检测」→ 查看OK/NG判定结果和缺陷框
5. 拖动「阈值」滑块调整检测灵敏度

### 用户权限
1. 左栏点击「👤 用户管理」
2. 输入 admin / admin123 → 点击「🔓 登录」
3. 登录后右侧面板激活 → 可新增/删除用户
4. 选中用户 → 右侧可编辑基本信息和页面权限
5. 点击「✅ 保存」→ 权限立即生效，左侧菜单会按权限实时显示/隐藏

## 常见问题

**Q: 运行时提示找不到smartmes.db？**  
A: 正常，首次连接数据库时自动创建于程序工作目录。

**Q: 数据库连接MySQL/SQL Server失败？**  
A: 本地未安装对应数据库服务。使用默认SQLite即可，无需安装任何数据库。

**Q: 视觉模块图像处理慢？**  
A: 纯C#实现，640×480约20ms，属正常范围。降低分辨率或提高检测间隔可改善。

**Q: 如何添加新页面模块？**  
A: 
1. 在 `SmartMES.UI/Modules/` 下新建文件夹
2. 创建 `XxxView.xaml` + `XxxViewModel.cs`
3. 在 `MainViewModel.cs` 添加属性和导航命令
4. 在 `MainWindow.xaml` 添加 DataTemplate 和导航按钮
5. 在 `UserModel.PagePermissions` 添加权限开关
