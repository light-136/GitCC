# SmartMES 工程功能分级与函数职责说明

> 日期: 2026-04-02  
> 目的: 帮助快速理解工程整体功能、分层职责、关键函数语义

---

## 1. 工程分级（从项目到模块）

### L1：解决方案级（`SmartMES.sln`）

1. `SmartMES.UI`：WPF 界面层（页面、导航、交互、状态展示）  
2. `SmartMES.Services`：系统服务层（日志、通信、自动化、设备服务）  
3. `SmartMES.Modules`：业务能力层（运动、视觉、数据库、文件、MES通信）  
4. `SmartMES.Core`：基础能力层（命令/通知、状态机、安全、调度、配方、追溯、IO、插件）  
5. `SmartMES.Tests`：单元测试层（核心逻辑自动化校验）

## 2. Core 层功能与关键函数

### 2.1 `Infrastructure`

- `ViewModelBase.OnPropertyChanged`：触发属性变化通知。
- `ViewModelBase.SetProperty`：统一属性赋值 + 变更通知。
- `RelayCommand.CanExecute/Execute`：命令可执行性判断和执行入口。
- `RelayCommand.RaiseCanExecuteChanged`：主动刷新按钮可用状态。

### 2.2 `StateMachine/StateMachineEngine`

- `AddTransition`：注册状态转换规则。
- `Fire`：按触发器进行状态流转（含守卫条件）。
- `ForceState`：强制切换状态（急停/初始化场景）。
- `BuildStandard`：生成标准工业状态机模板。

### 2.3 `Scheduler/TaskSchedulerService`

- `AddTask`：注册调度任务并计算首次执行时间。
- `StartAsync`：调度主循环（扫描到期任务并执行）。
- `ExecuteTaskAsync`：执行单任务并维护状态/错误。
- `Stop`：停止调度循环。

### 2.4 `Safety/SafetyService`

- `IsSafeToOperate`：检查全局 + 操作级互锁。
- `TriggerEStop`：触发急停并广播事件。
- `ResetEStop`：满足条件后复位急停。
- `AddInterlock`：动态增加互锁规则。

### 2.5 `Recipe/RecipeService`

- `GetAll/GetByName`：读取配方。
- `Activate`：切换当前活动配方。
- `SaveAsync/LoadAsync`：JSON 持久化。

### 2.6 `Traceability/TraceService`

- `StartTrace`：创建产品追溯主记录。
- `StartProcess`：创建工序记录。
- `EndProcess`：结束工序并写入结果。
- `AddParameter`：记录工艺参数。
- `Query`：按时间/结果检索追溯数据。

### 2.7 `IO/SimulatedIoDevice`

- `ReadInput/WriteOutput`：数字量读写。
- `ReadAnalog/WriteAnalog`：模拟量读写。
- `GetChannels`：获取全部通道快照。
- `SimulateInputs`：定时仿真输入变化。

## 3. Modules 层功能与关键函数

### 3.1 `MotionControl`

- `AxisController.Home/MoveTo/Pause/Resume/Stop/Reset`：单轴状态机控制。
- `AxisController.MoveToPosition`：插补核心（加减速 + 周期更新）。
- `MultiAxisController.HomeAllAsync`：多轴并行回零。
- `MultiAxisController.LinearInterpolateAsync`：多轴联动插补。
- `TenAxisController.RunGCodeAsync`：执行 G0/G1/G28/M0/M2 程序。
- `GCodeCommand.Parse`：解析单行 G 代码。

### 3.2 `Vision`

- `GenerateWorkpieceImage`：生成仿真工件图。
- `ToGrayscale`：灰度化。
- `SobelEdge`：边缘检测。
- `Inspect`：缺陷识别与结果判定。

### 3.3 `FileProcess`

- `ReadAsync/WriteAsync`（TXT/CSV/JSON/XML/Excel）：多格式统一读写接口。

### 3.4 `Database`

- `DbContext`：数据库连接与上下文管理。
- `Repositories`：仓储查询与统计聚合。

### 3.5 `MesComm`

- `HttpMesClient`/`OpcUaMesClient`/`MqttMesClient`：MES协议适配客户端。

### 3.6 `VisionMotion`

- `MultiStationScheduler`：视觉检测与运动任务协同调度。

---

## 4. Services 层功能与关键函数

### 4.1 日志/事件/配置

- `LoggingService.LogInfo/LogWarning/LogError/LogCommunication`：统一日志入口。
- `LoggingService.WriteToFileAsync`：异步落盘。
- `EventBusService`：跨模块事件发布订阅。
- `SettingsService.LoadAsync/SaveAsync`：配置读取与保存。

### 4.2 通信

- `TcpCommunicationService.ConnectAsync/SendAsync/ReceiveAsync`：TCP模拟通信。
- `SerialCommunicationService`：串口通信模拟/适配。

### 4.3 自动化

- `AutomationEngine.RunAsync`：流程引擎主执行。
- `AutomationEngine.Reset`：流程复位。

### 4.4 报警/设备

- `AlarmService`：报警触发、确认、清除、统计。
- `PlcDevice/SensorDevice`：设备状态模拟与读写操作。

---

## 5. UI 层功能与关键函数

### 5.1 主导航

- `MainViewModel`：页面导航、标题同步、用户状态、系统时钟。

### 5.2 关键业务页面

- `MotionViewModel`：三轴运动控制中心（全局控制 + 线性插补）。
- `Motion10AxisViewModel`：10轴程序执行、版面切换、状态可视化。
- `VisionViewModel`：摄像头模拟、检测触发、结果展示。
- `DatabaseViewModel`：连接、仿真、统计与导出。
- `FileViewModel`：文件加载、预览、导出。
- `UserViewModel`：用户登录、角色权限、页面访问控制。
