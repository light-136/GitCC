| 内存 | 4 GB | 8 GB+ |
| 磁盘空间 | 500 MB | 1 GB |
| 显示器 | 1920×1080 | 2560×1440 |
| 运行时 | .NET 9.0 Runtime | .NET 9.0 SDK |

## 2. 安装与启动

### 2.1 开发环境安装
```bash
# 1. 安装 .NET 9.0 SDK
# 下载地址：https://dotnet.microsoft.com/download/dotnet/9.0

# 2. 还原项目依赖
cd "SmartMES -V9.0"
dotnet restore

# 3. 编译项目
dotnet build -c Release

# 4. 运行应用
dotnet run --project SmartMES.UI
```

### 2.2 首次启动
1. 启动后进入登录界面
2. 默认管理员账号：admin / admin123
3. 默认操作员账号：operator / oper123
4. 默认观察者账号：observer / obs123
5. 登录后进入主界面仪表盘

## 4. 功能模块操作指南

### 4.1 仪表盘
- **实时数据**：显示设备连接状态、运行状态、产量统计
- **快速操作**：一键启动/停止生产、急停按钮
- **趋势图**：关键参数的实时趋势曲线

### 4.2 设备管理
- **设备列表**：查看所有已注册设备的状态
- **添加设备**：配置新设备的通信参数
- **设备详情**：查看设备的实时数据和历史记录
- **IO 仿真**：模拟 DI/DO/AI/AO 信号

### 4.3 通信监控
- **协议选择**：支持 TCP、串口、Modbus TCP、MQTT、OPC UA
- **连接管理**：建立/断开通信连接
- **数据收发**：手动发送数据帧，查看接收数据
- **通信日志**：记录所有通信数据包

#### Modbus TCP 操作
1. 输入从站 IP 和端口（默认 502）
2. 点击"连接"
3. 选择功能码（FC01 读线圈 / FC03 读寄存器 / FC05 写线圈 / FC06 写寄存器）
4. 输入起始地址和数量
5. 点击"读取"或"写入"

#### SECS/GEM 操作
1. 配置主机 IP 和端口（默认 5000）
2. 点击"连接" → HSMS TCP 连接建立
3. 点击"Select" → HSMS 握手完成
4. 点击"上线" → 发送 S1F13 建立通信
5. 查看消息日志中的 SECS-II 消息流
6. 可手动触发事件上报（S6F11）
7. 可查看状态变量（SV）和设备常量（EC）

### 4.4 运动控制

#### 单轴操作
1. 选择目标轴（X/Y/Z/A/B/C/U/V/W/S）
2. 设置速度和加速度参数
3. 操作：
   - **回零**：执行回零动作
   - **点动**：输入目标位置，点击"移动"
   - **暂停/恢复**：运动中可暂停和恢复
   - **急停**：立即停止所有运动

#### G 代码执行
1. 在 G 代码编辑器中输入程序
2. 支持的指令：
   - `G0 X_ Y_ Z_` — 快速定位
   - `G1 X_ Y_ Z_ F_` — 直线插补
   - `G2 X_ Y_ I_ J_ F_` — 顺时针圆弧
   - `G3 X_ Y_ I_ J_ F_` — 逆时针圆弧
   - `G4 P_` — 暂停（秒）
   - `G28` — 回零
   - `G54~G59` — 选择工件坐标系
   - `G90` — 绝对坐标模式
   - `G91` — 增量坐标模式
   - `M0` — 程序暂停
   - `M2` / `M30` — 程序结束
3. 点击"运行"执行程序
4. 运行中可"暂停"或"急停"

#### 运动规划选择
- **梯形曲线**：标准加减速，适合一般场景
- **S 曲线**：平滑加减速，适合精密定位（半导体设备推荐）

### 4.5 视觉检测

#### 图像采集
1. 选择相机（默认为模拟相机）
2. 设置曝光时间和增益
3. 点击"采集"获取图像

#### 检测流程
1. **自动检测**：点击"检测"，系统自动执行预设的检测管线
2. **管线配置**：
   - 添加处理步骤（灰度化 → 高斯模糊 → 阈值 → 形态学 → Blob分析）
   - 调整每步参数
   - 查看中间结果
3. **检测结果**：
   - OK/NG 判定
   - 缺陷位置和类型标注
   - 检测评分

#### 高级功能
- **模板匹配**：加载模板图像，在目标图像中搜索匹配位置
- **测量工具**：距离、角度、圆拟合、直线拟合
- **标定**：9 点标定，建立像素到世界坐标的映射

### 4.6 报警管理
- **实时报警**：显示当前活动报警
- **报警确认**：操作员确认已知报警
- **历史查询**：按时间、级别、来源筛选历史报警
- **报警级别**：
  - 信息（蓝色）：一般提示
  - 警告（黄色）：需要关注
  - 错误（红色）：需要处理
  - 紧急（红色闪烁）：需要立即处理

### 4.7 系统设置
- **用户管理**：添加/编辑/删除用户，分配角色
- **通信配置**：设备通信参数
- **系统参数**：日志级别、数据保留天数等
- **关于**：版本信息

## 5. 常见操作流程

### 5.1 标准生产流程
```
启动系统 → 登录 → 设备连接 → 回零 → 加载配方 → 启动生产 → 监控 → 停止 → 数据导出
```

### 5.2 SECS/GEM 通信流程
```
TCP连接 → Select握手 → S1F13建立通信 → S1F17请求上线 → 正常通信 → S6F11事件上报 → S2F41远程命令
```

### 5.3 视觉检测流程
```
相机连接 → 标定 → 配置管线 → 采集图像 → 执行检测 → 查看结果 → 导出报告
```

## 6. 故障排除

| 问题 | 可能原因 | 解决方法 |
|------|----------|----------|
| 启动失败 | .NET 运行时未安装 | 安装 .NET 9.0 Runtime |
| 通信连接失败 | IP/端口配置错误 | 检查网络配置和防火墙 |
| 运动不执行 | 轴处于错误状态 | 先执行"复位"再操作 |
| 视觉检测异常 | 参数不合适 | 调整阈值和管线参数 |
| 界面卡顿 | 数据刷新过快 | 降低刷新频率 |
