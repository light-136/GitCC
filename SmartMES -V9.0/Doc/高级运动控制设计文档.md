# SmartMES V9.0 高级运动控制设计文档

## 1. 概述

本文档描述 SmartMES V9.0 高级运动控制模块的设计方案，涵盖 S 曲线运动规划、圆弧插补、前瞻速度规划、电子齿轮/凸轮、坐标系变换、多策略回零等核心算法。

### 1.1 设计目标
- 支持半导体设备级别的精密运动控制
- 所有算法纯 C# 实现，可仿真运行
- 向后兼容现有 AxisController / MultiAxisController 架构
- 详细的中文注释，便于学习

### 1.2 模块架构
```
SmartMES.Core.Interfaces
    ├── IMotionProfile（运动规划接口）
    └── ICoordinateManager（坐标系管理接口）

SmartMES.Core.Models
    └── MotionModels（运动数据模型）

SmartMES.Modules.MotionControl
    ├── TrapezoidalProfile（梯形运动规划）

    ├── LookAheadPlanner（前瞻速度规划）
    ├── ElectronicGearing（电子齿轮/凸轮）

```

## 2. S 曲线运动规划

### 2.1 原理
S 曲线（加加速度限制运动）将运动分为 7 段，每段的加加速度（jerk）为常数。相比梯形曲线，S 曲线消除了速度的突变，使运动更加平滑，减少机械振动。

### 2.2 七段运动模型
```
加加速度(Jerk)
 ▲
 │  +J ┌───┐           ┌───┐
 │     │   │           │   │
 ├─────┘   └───────────┘   └─────→ 时间
 │             │   │
 │  -J         └───┘
 │
 │  T1   T2   T3  T4  T5  T6  T7

加速度(Acceleration)
 ▲
 │      ╱──╲
 │     ╱    ╲
 ├────╱      ╲────────╱──╲────→ 时间
 │                   ╱    ╲
 │                  ╱──────╲

速度(Velocity)
 ▲
 │        ┌──────────┐
 │       ╱            ╲
 │      ╱              ╲
 │     ╱                ╲
 │    ╱                  ╲
 │   ╱                    ╲
 └──╱──────────────────────╲──→ 时间
```

### 2.3 各段运动学公式

| 段 | 时间 | Jerk | 加速度 | 速度 | 位移 |
|----|------|------|--------|------|------|
| T1 | 加加速 | +J | a = J·t | v = ½J·t² | s = ⅙J·t³ |
| T2 | 匀加速 | 0 | a = a_max | v = v₁ + a·t | s = v₁·t + ½a·t² |
| T3 | 减加速 | -J | a = a_max - J·t | v = v₂ + a_max·t - ½J·t² | ... |
| T4 | 匀速 | 0 | a = 0 | v = v_max | s = v_max·t |
| T5 | 加减速 | -J | a = -J·t | v = v_max - ½J·t² | ... |
| T6 | 匀减速 | 0 | a = -a_max | v = v₅ - a·t | ... |
| T7 | 减减速 | +J | a = -a_max + J·t | v = v₆ - a_max·t + ½J·t² | ... |

### 2.4 短距离退化
当运动距离不足以达到最大速度时，自动退化：
- **去掉 T4（匀速段）**：加速后直接减速
- **去掉 T2 和 T6（匀加速/匀减速段）**：仅有加加速和减加速
- **极短距离**：仅有 T1 和 T7

## 3. 圆弧插补

### 3.1 I/J 圆心偏移法
```
G2 X_ Y_ I_ J_ F_    ; 顺时针圆弧
G3 X_ Y_ I_ J_ F_    ; 逆时针圆弧
```

圆心坐标计算：
```
Cx = Sx + I    ; 圆心X = 起点X + I偏移
Cy = Sy + J    ; 圆心Y = 起点Y + J偏移
```

角度计算：
```
θ_start = atan2(Sy - Cy, Sx - Cx)
θ_end   = atan2(Ey - Cy, Ex - Cx)
R = sqrt(I² + J²)
```

### 3.2 R 半径法
```
G2 X_ Y_ R_ F_
```
- 正 R：劣弧（小于180°）
- 负 R：优弧（大于180°）

圆心计算：
```
dx = Ex - Sx, dy = Ey - Sy
d = sqrt(dx² + dy²)
h = sqrt(R² - (d/2)²)
mx = (Sx + Ex) / 2, my = (Sy + Ey) / 2
Cx = mx ± h·dy/d
Cy = my ∓ h·dx/d
```

### 3.3 微线段生成
将圆弧按角度步长分解为直线段：
```
步长角度 = stepSize / R    ; stepSize 为线段长度（mm）
for θ = θ_start to θ_end step Δθ:
    x = Cx + R·cos(θ)
    y = Cy + R·sin(θ)
    生成 InterpolationPoint(x, y, feedRate)
```

## 4. 前瞻速度规划

### 4.1 三遍算法

#### 第一遍（前向）：计算拐角安全速度
```
for each junction between segment[i] and segment[i+1]:
    angle = 计算两段之间的夹角
    junction_vel = max_vel × cos(angle / 2)
    junction_vel = max(junction_vel, max_vel × 0.05)  ; 最低5%
```

#### 第二遍（后向）：确保减速可达
```
segment[last].exit_vel = 0  ; 最后一段出口速度为0
for i = last-1 downto 0:
    max_entry = sqrt(segment[i+1].entry_vel² + 2 × accel × distance[i])
    segment[i].exit_vel = min(junction_vel[i], max_entry)
```

#### 第三遍（前向）：确保加速可达
```
segment[0].entry_vel = 0  ; 第一段入口速度为0
for i = 0 to last:
    max_exit = sqrt(segment[i].entry_vel² + 2 × accel × distance[i])
    segment[i].exit_vel = min(segment[i].exit_vel, max_exit)
    segment[i+1].entry_vel = segment[i].exit_vel
```

### 4.2 速度曲线示例
```
速度
 ▲
 │     ╱╲    ╱──╲
 │    ╱  ╲  ╱    ╲
 │   ╱    ╲╱      ╲
 │  ╱               ╲
 │ ╱                 ╲
 └──────────────────────→ 路径
   段1  拐角  段2  拐角  段3
```

## 5. 电子齿轮与凸轮

### 5.1 电子齿轮
从轴位置 = 主轴位置 × 齿轮比 + 相位偏移
```
slave_target = master.Position × ratio + phaseOffset
```
10ms 周期后台线程持续跟踪。

### 5.2 电子凸轮
凸轮表定义主从轴的非线性对应关系：
```
CamPoint[0]: master=0,   slave=0
CamPoint[1]: master=90,  slave=50
CamPoint[2]: master=180, slave=100
CamPoint[3]: master=270, slave=50
CamPoint[4]: master=360, slave=0
```

查找算法：线性插值
```
找到 master 所在的区间 [P[i], P[i+1]]
t = (master - P[i].master) / (P[i+1].master - P[i].master)
slave_target = P[i].slave + t × (P[i+1].slave - P[i].slave)
```

## 6. 坐标系变换

### 6.1 坐标系层次
```
机械坐标系（Machine）
    │
    ├── G54 工件坐标系1（偏移 + 旋转）
    ├── G55 工件坐标系2
    ├── G56 工件坐标系3
    ├── G57 工件坐标系4
    ├── G58 工件坐标系5
    └── G59 工件坐标系6
         │
         └── 工具坐标系（Tool，叠加在工件坐标系上）
```

### 6.2 变换公式
机械坐标 → 工件坐标：
```
wx = (mx - offsetX) × cos(θ) + (my - offsetY) × sin(θ)
wy = -(mx - offsetX) × sin(θ) + (my - offsetY) × cos(θ)
wz = mz - offsetZ
```

工件坐标 → 机械坐标（逆变换）：
```
mx = wx × cos(θ) - wy × sin(θ) + offsetX
my = wx × sin(θ) + wy × cos(θ) + offsetY
mz = wz + offsetZ
```

## 7. 多策略回零

### 7.1 传感器回零（SensorBased）
```
快速负方向搜索 → 触发原点传感器 → 反向慢速搜索 → 离开传感器边缘 → 设置零点
```

### 7.2 编码器脉冲回零（IndexPulse）
```
快速搜索传感器 → 慢速搜索编码器Z脉冲 → 设置零点（精度最高）
```

### 7.3 硬限位回零（HardStop）
```
慢速负方向运动 → 检测堵转（位置不变化）→ 后退少量 → 设置零点
```

### 7.4 当前位置回零（CurrentPosition）
```
直接将当前位置设为零点 + 偏移量（即时完成）
```

## 8. 扩展 G 代码

### 8.1 新增指令
| 指令 | 功能 | 参数 |
|------|------|------|
| G2 | 顺时针圆弧 | X Y I J F 或 X Y R F |
| G3 | 逆时针圆弧 | X Y I J F 或 X Y R F |
| G4 | 暂停 | P（秒） |
| G17 | XY 平面选择 | — |
| G18 | XZ 平面选择 | — |
| G19 | YZ 平面选择 | — |
| G54~G59 | 工件坐标系选择 | — |
| G90 | 绝对坐标模式 | — |
| G91 | 增量坐标模式 | — |
| M30 | 程序结束并复位 | — |

### 8.2 模态状态
G 代码解析器维护以下模态状态：
- 当前坐标系（默认 Machine）
- 绝对/增量模式（默认 G90 绝对）
- 当前平面（默认 G17 XY）
- 当前进给速率

### 8.3 示例程序
```gcode
; SmartMES V9.0 高级运动控制示例
G28                         ; 全轴回零
G54                         ; 选择工件坐标系1
G90                         ; 绝对坐标模式
G0 X0 Y0 Z50               ; 快速移动到安全高度
G1 X100 Y0 Z0 F3000        ; 直线插补到(100,0)
G2 X0 Y100 I-100 J0 F2000  ; 顺时针圆弧到(0,100)
G3 X-100 Y0 I0 J-100 F2000 ; 逆时针圆弧到(-100,0)
G1 X0 Y0 F2000             ; 返回原点
G4 P1.0                    ; 暂停1秒
G91                         ; 切换增量模式
G1 Z-30 F1000              ; Z轴下降30mm
G90                         ; 切换回绝对模式
G0 Z50                     ; Z轴抬起
M30                         ; 程序结束
```

## 11. 30轴工业级运动控制系统

V9.0+ 新增了面向半导体行业的30轴工业级运动控制系统，详细设计请参见独立文档：

**→ [30轴运动控制系统设计文档](30轴运动控制系统设计文档.md)**

该系统在本文档描述的核心模块基础上，增加了：
- **轴组管理（AxisGroupManager）**：将30轴组织为6个运动组
- **多通道控制（MultiChannelController）**：6个独立G代码通道并行执行
- **碰撞检测（CollisionDetector）**：后台50ms周期实时检测
- **性能监控（PerformanceMonitor）**：跟随误差/周期时间/负载率采样
- **ThirtyAxisController**：统一编排的顶层门面
