# SmartMES V9.0 项目概要说明

## 1. 项目概述

### 1.1 项目名称
SmartMES V9.0 — 智能制造执行系统（工业上位机平台）

### 1.2 项目定位
SmartMES V9.0 是一个面向半导体、精密制造等行业的工业上位机软件平台。系统采用模块化架构设计，集成了运动控制、视觉检测、SECS/GEM 半导体通信、MES 对接、设备管理等核心功能，可作为工业自动化学习平台和实际项目的开发基础。

### 1.3 技术栈
| 层次 | 技术选型 | 说明 |
|------|----------|------|
| 运行时 | .NET 9.0 (Windows) | 最新 LTS 版本 |
| UI 框架 | WPF + XAML | 桌面端工业界面 |
| 架构模式 | MVVM | 视图与逻辑分离 |
| 语言 | C# 13 / C++ (Native) | 主语言 + 高性能计算 |
| 测试框架 | xUnit | 单元测试 |
| 构建工具 | MSBuild / CMake | .NET + Native 双构建 |

## 2. 系统架构

### 2.1 分层架构
```
┌─────────────────────────────────────────────┐
│                SmartMES.UI                   │
│         WPF 界面层（MVVM ViewModels）         │
├─────────────────────────────────────────────┤
│              SmartMES.Modules                │
│    功能模块层（运动控制/视觉/MES通信/数据库）    │
├─────────────────────────────────────────────┤
│              SmartMES.Services               │
│    服务层（通信/报警/日志/事件总线/SECS-GEM）    │
├─────────────────────────────────────────────┤
│               SmartMES.Core                  │
│      核心层（接口/模型/状态机/安全/配方）        │
├─────────────────────────────────────────────┤
│              SmartMES.Native                 │
│         原生层（C++ 高性能计算）               │
└─────────────────────────────────────────────┘
```

### 2.2 项目依赖关系
```
SmartMES.UI ──→ SmartMES.Modules ──→ SmartMES.Core
     │                │
     └──→ SmartMES.Services ──→ SmartMES.Core
                      │
               SmartMES.Native
```

### 2.3 核心设计原则
1. **仿真优先**：所有模块均可在无硬件环境下运行和测试
2. **接口驱动**：Core 层定义接口，Services/Modules 层提供实现
3. **事件解耦**：模块间通过 IEventBus 通信，避免直接依赖
4. **线程安全**：所有共享状态使用 lock 保护
5. **中文注释**：全部代码注释使用中文，便于学习

## 4. 关键技术特性

### 4.1 运动控制
- **S曲线运动规划**：7段式加加速度限制运动，消除速度突变
- **圆弧插补**：G2/G3 指令，支持 I/J 圆心偏移和 R 半径两种模式
- **前瞻速度规划**：三遍算法优化路径拐角速度
- **电子齿轮/凸轮**：主从轴同步跟踪
- **坐标系变换**：G54~G59 工件坐标系 + 工具坐标
- **多策略回零**：传感器/编码器脉冲/硬限位/当前位置

### 4.2 SECS/GEM 半导体通信
- **HSMS 传输层**（SEMI E37）：TCP 传输、Select/Deselect/Linktest 控制消息、T3~T8 定时器
- **SECS-II 编解码**（SEMI E5）：递归 SecsItem 树、大端字节序、SML 文本表示
- **GEM 合规层**（SEMI E30）：双层状态机、SV/EC/CE 管理、告警、远程命令、工艺程序
- **模拟主机**：完整的 GEM Host 仿真器，用于测试和学习

### 4.3 视觉系统
- **图像处理管线**：可配置的多步处理链
- **模板匹配**：NCC 归一化互相关 + 图像金字塔加速
- **Blob 分析**：两遍连通域标记 + 特征提取
- **形态学运算**：腐蚀/膨胀/开运算/闭运算/顶帽/黑帽
- **直方图分析**：OTSU 自动阈值、直方图均衡化、自适应阈值
- **测量工具**：距离/角度/圆拟合/直线拟合
- **9点标定**：像素到世界坐标的仿射变换

## 6. 快速开始

```bash
# 克隆项目
git clone <repository-url>

# 还原依赖
cd "SmartMES -V9.0"
dotnet restore

# 编译
dotnet build -c Release

# 运行测试
dotnet test

# 启动应用
dotnet run --project SmartMES.UI
```
