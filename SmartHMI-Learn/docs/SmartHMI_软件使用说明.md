# SmartHMI 工业上位机系统 — 软件使用说明

**版本：** 1.0  
**适用人员：** 操作员、工程师、管理员

---

## 1. 系统要求

| 项目 | 要求 |
|------|------|
| 操作系统 | Windows 10/11 (x64) |
| .NET 运行时 | .NET 9.0 Desktop Runtime |
| 内存 | 最低 512 MB，推荐 2 GB |
| 分辨率 | 最低 1280×600 |

---

## 2. 启动与登录

### 2.1 启动程序

运行 `SmartHMI.UI.exe`，程序启动后显示登录界面。

### 2.2 默认账号

| 用户名 | 密码 | 角色 | 权限说明 |
|--------|------|------|----------|
| admin | admin123 | 管理员 | 全部功能 |
| engineer | eng123 | 工程师 | 配置+操作 |
| operator | op123 | 操作员 | 监控+操作 |
| viewer | view123 | 查看者 | 只读 |

### 2.3 登录步骤

1. 在"用户名"输入框输入账号
2. 在"密码"输入框输入密码
3. 点击"登录"按钮或按 Enter 键
4. 登录成功后自动跳转到仪表盘

## 4. 各功能模块说明

### 4.1 仪表盘（Dashboard）

实时显示系统关键指标：

- **在线设备数**：当前处于 Online 状态的设备数量
- **活动报警数**：未处理的报警条数（红色显示）
- **温度 AI0**：模拟量输入通道 0 的当前值（°C）
- **压力 AI1**：模拟量输入通道 1 的当前值（bar）
- **数字输入状态**：16 路 DI 通道实时状态（绿色=ON，灰色=OFF）
- **设备状态列表**：各设备当前状态一览
- **最近报警**：最新触发的报警记录

### 4.2 设备管理（Device）

管理所有接入设备：

- **设备列表**：显示设备名称、类型、IP、端口、状态、最后更新时间
- **状态指示灯**：绿色=在线，灰色=离线，红色=故障，黄色=连接中
- **连接设备**：对选中设备发起连接
- **断开连接**：断开选中设备的连接
- **刷新**：手动刷新设备状态
- **设备详情**：点击设备行，下方显示详细信息

### 4.3 通信监控（Communication）

监控所有通信通道：

- **通道列表**（左侧）：显示 TCP、串口、MQTT、OPC UA 四条通道
- **通道信息**：名称、协议类型
- **流量统计**：发送/接收字节数
- **消息日志**：实时显示收发的原始消息（绿色 Consolas 字体）
- **发送面板**：在输入框输入内容，点击"发送"向当前通道发送数据

### 4.4 运动控制（Motion）

控制 X/Y/Z 三轴运动：

- **轴状态**：显示当前轴状态（Disabled/Enabled/Homing/Idle/Moving 等）
- **当前位置**：实时显示轴位置（mm）
- **使能/禁用**：开启或关闭轴使能
- **回零**：执行回零操作
- **绝对定位**：输入目标位置，点击"移动"执行绝对定位
- **点动**：按住正向/负向点动按钮进行手动移动
- **急停**：立即停止所有运动（红色按钮）

### 4.5 报警管理（Alarm）

处理系统报警：

- **活动报警**标签页：显示所有未清除的报警
  - 报警代码、消息、级别、来源、触发时间
  - **确认**：标记报警已被操作员知晓
  - **清除**：清除单条报警
  - **全部清除**：清除所有活动报警
- **历史报警**标签页：查看已清除的历史报警记录

报警级别颜色：
- 🔵 Info — 信息
- 🟡 Warning — 警告（黄色）
- 🔴 Error — 错误（红色）
- 🔴 Critical — 严重（深红色）

### 4.6 系统日志（Log）

查看系统运行日志：

- **模块筛选**：按模块名称过滤日志
- **级别筛选**：按日志级别（Debug/Info/Warning/Error）过滤
- **日志列表**：时间戳、级别、模块、消息内容
- 最多保留 2000 条日志，超出后自动滚动丢弃最旧的

### 4.7 参数配置（Settings）

修改系统运行参数：

- **数据刷新间隔**：UI 数据刷新频率（毫秒）
- **最大报警数**：内存中保留的最大报警条数
- **数据库路径**：SQLite 数据库文件路径
- **日志级别**：最低记录的日志级别
- 修改后点击"保存"生效，点击"重置"恢复默认值

### 4.8 用户管理（User）

管理系统用户（需管理员权限）：

- **用户列表**：显示所有用户的账号、角色、状态、最后登录时间
- **新增用户**：填写用户名、密码、角色后点击"添加"
- **启用/禁用**：切换用户账号的激活状态
- **删除用户**：删除选中用户（不可删除当前登录用户）

## 6. 常见问题

**Q：登录后界面空白？**  
A：检查 App.xaml 中是否正确配置了 DataTemplate ViewLocator。

**Q：设备状态一直显示 Offline？**  
A：当前为仿真模式，设备状态由 DeviceManager 随机模拟，属正常现象。

**Q：报警徽章不消失？**  
A：进入"报警管理"页面，点击"全部清除"清除所有活动报警。

**Q：运动控制轴无法使能？**  
A：确认轴当前状态为 Disabled，点击"使能"按钮后状态应变为 Enabled。
